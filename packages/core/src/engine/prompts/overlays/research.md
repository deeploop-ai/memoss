## Schema pack overlay: research

- Organize durable concepts under `topics/` — one concept per page.
- Maintain a synthesis page (e.g. `topics/thesis.md` or vault-specific path in instructions) when doing long-running research.
- Survey papers: update thesis + per-topic pages + cross-links; avoid duplicating abstract text on every page.
- Put reusable definitions under `references/` only when reference mint gates pass.
- Save exploratory Q&A under `notes/` as `type: Note`.
